Overview

The kirby-item-sliding element is used to extend the kirby-item component with the functionality to reveal buttons when swiped. These buttons are created from a list of swipeAction objects which is provided via the swipeActions input property.

Each swipeAction object can be given properties to configure the corresponding button with a title, icon, type, disabled status and behaviour when selected.
