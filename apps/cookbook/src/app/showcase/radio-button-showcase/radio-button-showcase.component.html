The purpose of a radio button is to select either 0-1 (zero to one) values, amongst a set of options.

This behaviour is very similar to that of the Dropdown, but where as a dropdown can contain several options, a radio button is usually only used where a limited set is options are to be selected amongst.

Simple

This example is using the kirby-radio-button-group and kirby-radio-button Components by binding to the components using simple property (@Input) and event bindings.

Template-driven Forms

This example is using the kirby-radio-button-group and kirby-radio-button Components by binding the components using Angular's template-driven forms ([ngModel]="...", (ngModelChange)="..." or [(ngModel)]="...").

Reactive Forms

This example is using the kirby-radio-button-group and kirby-radio-button Components by binding the components using Angular's reactive forms (formControlName="..." or [formControl]="...")
