The Text Link component redirects the user to a new destination that can either be a route within the application or an external web page.

If the component is supplied with an internal route, the Angular RouterLink directive is used to avoid reloading the entire application. External links will automatically get an icon to indicate that the user is sent to an external web page when clicking the link.

External links will open in a new tab by default, but this behavior can be changed via the target property, similar to the target attribute of a .
